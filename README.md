# Adaptive-MutaSwarmClus

## Overview

Adaptive-MutaSwarmClus is a hybrid bio-inspired clustering algorithm developed as a final year project. It combines multiple optimization techniques to improve clustering performance on complex datasets.

---

## Key Features

* Combines ACO, MBF, Levy Flight and ILS
* Handles noisy and high-dimensional data
* Uses multiple clustering evaluation metrics
* Includes comparison with MBF and K-Means

---

## Algorithms

* Adaptive-MutaSwarmClus (Proposed)
* MBF (Baseline)
* K-Means (Standard)

---

## Datasets

* Iris, Wine, Breast Cancer
* Seeds, Thyroid, Ionosphere
* Image Segmentation, Vote
* Dermatology, Vehicle

---
